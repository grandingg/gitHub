## Мои знания о Git

1. Чтобы создать репозиторий в папке, нужно ввести компанду:

```
git init
```

- Если мы добавляем новый файл в папке, то нужно ещё добавить его в git:

```
git add .
```

2. Далее, после каждого изменения в файлах папки, нужно вводить так называемый коммит, для это мы пользуемся командой:

```
git commit -m "Что-то тут пишем"
```

- Но, если мы вдруг не написали в "нужный текст" и забыли про (-m), то может вылезти встроенный редактор vim, чтобы из него выйти надо ввести компанду :q! и нажать Enter

3. После создания коммита, мы можем проверить, что у нас там записалось, для этого введём команду:

```
git status
```

4. Теперь добавим наш commit в репозиторий, но для начала его сохраним так сказать, для этого нажмём команду:

```
git add .
```

5. Теперь "пушним" это всё в наш гид, если мы делаем первый раз в этом файле, то вводим команду:

```
git push -u origin main
```

- Если уже не первый раз, то только:

```
git push
```

Готово, мы всё добавили!
