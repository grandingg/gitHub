## Мои знания о Git

- Чтобы создать репозиторий в папке, нужно ввести компанду:

```
git init
```

- Если мы добавляем новый файл в папке, то нужно ещё добавить его в git:

```
git add .
```

- Далее, после каждого изменения в файлах папки, нужно вводить так называемый коммит, для это мы пользуемся командой:

```
git commit -m "Что-то тут пишем"
```

- Но, если мы вдруг не написали в "нужный текст" и забыли про (-m), то может вылезти встроенный редактор vim, чтобы из него выйти надо ввести компанду :q! и нажать Enter

- После создания коммита, мы можем проверить, что у нас там записалось, для этого введём команду:

```
git status
```

- Теперь добавим наш commit в репозиторий, но для начала его сохраним так сказать, для этого нажмём команду:

```
git add .
```

- Нам нужно вставить наш коммит в main branch, потому что сейчас у нас branch \*, мы сначала проверим, а потом добавим main

```
git branch
git branch -M main
```

- Сейчас соединим наш удалённый репозиторий, для начала создаём на git hub новый репозиторий с таким же названием как наша папка, потом копируем SSH ссылку и теперь в нашем терминале вводим:

```
git remote add origin git@github.com:ТВОЙ_НИК/ИМЯ_РЕПОЗИТОРИЯ.git
```

- Проверим подключение:

```
git remote -v
```

- Должнл выйти что-то типо:

```
origin  git@github.com:grandingg/gitHub.git (fetch)
origin  git@github.com:grandingg/gitHub.git (push)
```

- Теперь "пушним" это всё в наш гид, если мы делаем первый раз в этом файле, то вводим команду:

```
git push -u origin main
```

- Если уже не первый раз, то только:

```
git push
```
